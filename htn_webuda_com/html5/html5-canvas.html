Thẻ canvas trong HTML5

HTML5 Canvas Element Tutorial

Tag Canvas là 1 tag HTML tương tự như <div>,<a>, hoặc <table>. Điểm khác biệt là nội dung của nó được chuyển tải qua javascript. Để Sử dụng tag <canvas>, cần đặt tag <canvas> trong file html. Nội dung được hiển thị ra qua javascript. Tạo Context. Và sử dụng HTML5 Canvas API để vẽ

Khi sử dụng canvas, điều quan trọng là phải hiểu sự khác biệt giữa tag <canvas> và canvas context. Canvas context là một object với các thuộc tính, và phương thức mà bạn có thể sử dụng để làm đồ họa bên trong tag <canvas>. context có thể là 2d hay WebGL (3D).

Mỗi phần tử chỉ có 1 context. Nếu bạn sử dụng nhiều getContext() nhiều lần nó sẽ trả về một tham chiếu đến các đối tượng context giống nhau.

Cấu trúc thẻ canvas

<body>
<canvas id="myCanvas" width="578" height="200"></canvas>
<script>
var canvas = document.getElementById('myCanvas');
var context = canvas.getContext('2d');
// do stuff here
</script>
</body>
