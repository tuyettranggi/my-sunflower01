Bố cục HTML5

Các trình duỵêt không hổ trợ các thẻ section,article, hgroup, header, footer, nav, aside...

→ Để fix cho các trình duỵêt dùng đoạn css:
display:bock và js

 (function (){
	var els = [	'section', 'article', 'hgroup', 'header', 'footer', 'nav', 'aside', 'figure', 'mark', 'time', 'ruby', 'rt', 'rp' ];
	for (var i=0; i<els.length; i++){
		  document.createElement(els[i]);
		}
})();

Các tag bố cục html

  • <header> : bố trị nội dung phần header
  • <section> : bố trí #main
  • <article> : bố trí phần nội dung riêng biệt
  • <aside> : siderbar
  • <nav> : bố trí các menu, link navigation
