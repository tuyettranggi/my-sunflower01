Chuyển đổi source code thành html

Convert source code to HTML

Code bình thường:

<!DOCTYPE HTML>
<html>
<head>
<style>
body {
margin: 0px;
padding: 0px;
}
</style>
</head>
<body>
<canvas id="myCanvas" width="578" height="200"></canvas>
<script type="text/javascript">
var line1 = document.getElementById('ex1').getContext('2d');
line1.beginPath();
line1.moveTo(0,0);
line1.lineTo(360,180);
line1.stroke();
</script>
</body>
</html>

Sử dụng: syntaxHighlighter

Tham khảo: http://alexgorbatchev.com/SyntaxHighlighter/manual/brushes/xml.html
