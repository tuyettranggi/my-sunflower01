HTML5 Canvas Path Tutorial

Tìm tọa độ cho thẻ canvas

Để tạo đường path với HTML CANVAS, chúng ta có thể kết nối nhiều đưòng path phụ với nhau. Điểm kết thúc của mổi đường path phụ, sẽ cho điểm bắt đầu mới. We can use the lineTo(), arcTo(), quadraticCurveTo(), and bezierCurveTo() methods to construct each subpath which makes up our path. We can also use the beginPath() method each time we want to start drawing a new path.

more

var c = document.getElementById(“myCanvas”);
var context = c.getContext(“2d”);
context.beginPath();
//... path drawing operations
context.closePath();

fillRect(x,y,width,height) Vẽ một hình chữ nhật được tô
 
strokeRect(x,y,width,height) vẽ một khung chữ nhật  
clearRect(x,y,width,height) xóa một vùng và làm vùng đó transparent  
arc(x, y, radius, startAngle, endAngle, anticlockwise) vẽ một cung tròn radius: bán kính
startAngle: góc bắt đầu
endAngle: góc kết thúc
anticlockwise: ngược hay cùng chiều kim đồng hồ. sử dụng giá trị false , true		 
moveTo(x,y);
